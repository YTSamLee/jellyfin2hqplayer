==================================================
Jellyfin2HQPlayer 插件 1.3.0
==================================================

支持 Jellyfin 插件库部署

该插件完全支持官方的 Jellyfin 插件安装和部署流程。

插件功能：

- 音频文件路径 → Jellyfin ItemId 映射
- 为 Jellyfin 音频库提供 REST API 查询路由，以便 Jellyfin2HQPlayer 集成。

GitHub 仓库：  
https://github.com/YTSamLee/Jellyfin2HQPlayer-plugin

安装指南：

1. 打开 Jellyfin 管理面板 → 插件
2. 点击右上角的按钮：**管理仓库**
3. 添加仓库：
   - 仓库名称：Jellyfin2HQPlayer
   - 仓库 URL：https://raw.githubusercontent.com/YTSamLee/Jellyfin2HQPlayer-plugin/main/manifest.json
4. 保存并刷新 Jellyfin 页面
5. 返回插件页面
6. 在“可用插件”下找到：**Jellyfin2HQPlayer**
7. 点击安装
8. 安装完成后重启 Jellyfin

验证：

1. 打开管理面板 → 插件 → 我的插件 → **Jellyfin2HQPlayer**
2. 插件设置页面显示：
   - 插件已准备好状态
   - 索引音频文件的数量
   - 总音频文件数量
   - 上次更新的时间
   - 重新构建索引按钮
