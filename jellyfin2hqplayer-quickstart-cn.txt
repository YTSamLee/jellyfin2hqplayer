==================================================
Jellyfin2HQPlayer 部署指南
==================================================

1. Linux x64
==================================================

所需文件：

- jellyfin2hqplayer-linux-x64-x.x.x.tar.gz
- deploy-jellyfin2hqplayer-linux-x64.sh

运行：

    chmod +x deploy-jellyfin2hqplayer-linux-x64.sh
    sudo ./deploy-jellyfin2hqplayer-linux-x64.sh

（可选）自定义端口：

    sudo PORT=8080 ./deploy-jellyfin2hqplayer-linux-x64.sh

==================================================
2. macOS arm64
==================================================

所需文件：

- jellyfin2hqplayer-macos-arm64-x.x.x.tar.gz
- deploy-jellyfin2hqplayer-macos-arm64.sh

在本机终端（Terminal）中执行：

    chmod +x deploy-jellyfin2hqplayer-macos-arm64.sh
    ./deploy-jellyfin2hqplayer-macos-arm64.sh

注意：

- 不要使用 sudo

（可选）指定端口：

    PORT=8080 ./deploy-jellyfin2hqplayer-macos-arm64.sh

--------------------------------------------------
说明：启动方式
--------------------------------------------------

macOS 使用 LaunchAgent 机制：

- 用户登录后自动启动（登录启动）

==================================================
3. Windows x64
==================================================

所需文件：

- jellyfin2hqplayer-windows-x64-x.x.x.rar

将文件包解压到任意目录：

例如：

    C:\jellyfin2hqplayer

--------------------------------------------------
3.1 直接运行
--------------------------------------------------

进入解压后的目录，直接运行：

    start-jellyfin2hqplayer.ps1

或者：

    jellyfin2hqplayer-win-x64.exe

--------------------------------------------------
3.2 配置登录后自动启动（推荐）
--------------------------------------------------

以管理员身份打开 PowerShell。

进入解压后的目录：

    cd C:\jellyfin2hqplayer

执行：

    .\Create-Jellyfin2HQPlayerTask-AtLogon.ps1

特点：

- 用户登录后自动启动
- 不需要 Windows 密码
- 推荐默认方案
- 最适合 Jellyfin2HQPlayer

--------------------------------------------------
3.3 配置 Windows 开机后台启动（高级）
--------------------------------------------------

以管理员身份打开 PowerShell。

进入解压后的目录：

    cd C:\jellyfin2hqplayer

执行：

    .\Create-Jellyfin2HQPlayerTask-AtStartup.ps1

特点：

- Windows 开机自动后台启动
- 即使用户未登录也会运行
- 需要输入 Windows 密码
- 更接近后台服务模式

--------------------------------------------------
3.4 （可选）自定义端口
--------------------------------------------------

通过图形界面添加系统环境变量：

    PORT=8080

或者通过命令行设置系统环境变量
（需要管理员权限）：

    setx PORT "8080" /M

==================================================
3.5 重要提示
--------------------------------------------------

无论是**直接运行**、**配置自动启动**，还是**配置后台启动**，都可能会遇到 **防火墙提示**。

首次运行时，请确保点击：

✔ 允许访问（专用网络）

否则 Web 界面可能无法正常访问。

--------------------------------------------------
建议：手动运行一次
--------------------------------------------------

配置自动启动后，**建议先手动运行一次**，以确保配置生效并正常启动。

执行以下命令，手动启动 Jellyfin2HQPlayer：

    jellyfin2hqplayer-win-x64.exe

或者：

    start-jellyfin2hqplayer.ps1

手动运行一次后，**系统将确保下次登录时自动启动**。

==================================================
4. 访问 Web 界面
==================================================

访问地址：

    http://<IP>:3000

首次启动：

- 默认端口：3000（或自定义 PORT）
- 登录 Jellyfin 服务器（Login）
- 选择 HQPlayer 设备（Play To）
