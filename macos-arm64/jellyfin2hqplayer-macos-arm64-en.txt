# Jellyfin2HQPlayer Deployment (macOS arm64)

1. Prepare package
- Option 1: Download the release package directly to the target machine
On the target machine, use `wget` or `curl` to download the release package. For example:

wget https://github.com/YTSamLee/jellyfin2hqplayer/releases/download/v1.3.0/jellyfin2hqplayer-macos-arm64-1.3.0.tar.gz
wget https://github.com/YTSamLee/jellyfin2hqplayer/releases/download/v1.3.0/deploy-jellyfin2hqplayer-macos-arm64.sh
# or
curl -LO https://github.com/YTSamLee/jellyfin2hqplayer/releases/download/v1.3.0/jellyfin2hqplayer-macos-arm64-1.3.0.tar.gz
curl -LO https://github.com/YTSamLee/jellyfin2hqplayer/releases/download/v1.3.0/deploy-jellyfin2hqplayer-macos-arm64.sh

- Option 2: Manually transfer the file to the target machine
Alternatively, you can upload the release package to the target machine using SCP, FTP, or any other preferred file transfer method. Once the file is on the target machine, proceed to the next steps.

2. Deploy
Make the script executable:
chmod +x deploy-jellyfin2hqplayer-macos-arm64.sh

Auto-select latest package:
./deploy-jellyfin2hqplayer-macos-arm64.sh

Or specify package:
./deploy-jellyfin2hqplayer-macos-arm64.sh jellyfin2hqplayer-macos-arm64-*.tar.gz

3. Port
Default:
3000

Specify port during deploy:
PORT=8080 ./deploy-jellyfin2hqplayer-macos-arm64.sh

4. Change Port (after deploy)
Edit plist:
nano ~/Library/LaunchAgents/com.jellyfin2hqplayer.plist

Find:
<key>PORT</key>
<string>3000</string>

Change to:
<string>8080</string>

Then reload:
launchctl bootout gui/$(id -u)/com.jellyfin2hqplayer
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.jellyfin2hqplayer.plist
launchctl kickstart -k gui/$(id -u)/com.jellyfin2hqplayer

5. Access
http://<IP>:PORT

6. If not accessible
Check service:
launchctl print gui/$(id -u)/com.jellyfin2hqplayer

Check logs:
tail -f /tmp/jellyfin2hqplayer.log
tail -f /tmp/jellyfin2hqplayer.err

7. macOS Permission
If blocked in background:
System Settings → General → Login Items → Allow jellyfin2hqplayer

8. Stop / Restart
Restart:
launchctl kickstart -k gui/$(id -u)/com.jellyfin2hqplayer

Stop:
launchctl bootout gui/$(id -u)/com.jellyfin2hqplayer
