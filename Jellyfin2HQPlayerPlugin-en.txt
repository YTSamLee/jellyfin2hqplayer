==================================================
Jellyfin2HQPlayer Plugin 1.3.0
==================================================

Supports Jellyfin Plugin Repository Deployment

This plugin fully supports the official Jellyfin plugin installation and deployment workflow.

Plugin Features:

- Audio file path → Jellyfin ItemId mapping
- Provides REST API query routes for Jellyfin audio libraries, enabling Jellyfin2HQPlayer integration.

GitHub Repository:  
https://github.com/YTSamLee/Jellyfin2HQPlayer-plugin

Installation Guide:

1. Open the Jellyfin Dashboard → Plugins
2. Click the button at the top-right: **Manage Repositories**
3. Add the repository:
   - Repository Name: Jellyfin2HQPlayer
   - Repository URL: https://raw.githubusercontent.com/YTSamLee/Jellyfin2HQPlayer-plugin/main/manifest.json
4. Save and refresh the Jellyfin page
5. Return to the Plugins page
6. Under "Available Plugins," find: **Jellyfin2HQPlayer**
7. Click Install
8. After installation, restart Jellyfin

Verification:

1. Open the Dashboard → Plugins → My Plugins → **Jellyfin2HQPlayer**
2. The plugin settings page will show:
   - Plugin READY status
   - Indexed audio file count
   - Total audio file count
   - Last updated time
   - Rebuild index button
